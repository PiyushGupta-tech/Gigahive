All website files are in this folder

If files don’t show in Finder, press Cmd + R to refresh.

This folder contains:

  • 18 HTML pages (index, login, cart, category pages…)
  • css/styles.css
  • js/app.js, js/products.js
  • README.md, AUTO.md, FILES_MANIFEST.txt

To run the site: open Terminal, cd to this folder, then run:
python3 -m http.server 8080
Then open http://localhost:8080
